A faithful terminal simulation of the WOPR computer from the
1983 film **WarGames** — CRT aesthetics, variable-speed typewriter
text, single HTML file, zero dependencies.

**Launch:**

    open index.html

Or serve locally:

    python3 -m http.server 8080
